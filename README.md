# Differential Network creation App
This is an application to create a differential network from two edgelist.

## System Requirements
- MacOS Catalina 10.15.2
- R 3.6.0
  - shiny 1.5.0
  - rsconnect 0.8,16
  - igraph 1.2.4.1
  - DT 0.15
  - dplyr 1.0.1

### Step 1. Clone this repository to local.
### Step 2. Install R and each packages
